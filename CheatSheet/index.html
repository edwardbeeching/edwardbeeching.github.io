Semantic HTML Reference

Semantic Tags

Tag Name Description
<header> Header An element containing introductory content.
<main> Main An element containing the page's primary content.
<footer> Footer An element found at the end of a block of content.
<nav> Navigation An element containing a block of navigation links.
<section> Section An element containing other elements of the same theme.
<article> Article An element of self-contained content, such as a post.
<figure> Figure An element containing media, such as an image.
<figcaption> Figure Caption An element describing content found in a <figure>.
<aside> Aside An element containing related content, such as links.
